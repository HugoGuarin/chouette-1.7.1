Pour le déploiement d'un site tatrobus (à partir du chouette_XXXX.tar.gz) :
- copier chouette.war dans tatrobus.war.
- créer un répertoire /etc/chouette/tatrobus
- copier les autres fichiers de l'archive chouette_XXXX.tar.gz (fichiers de conf)
  dans le répertoire /etc/chouette/tatrobus
- déployer tatrobus.war dans tomcat6.0
- ça marche
